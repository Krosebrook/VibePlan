# Agentic OS: The Ghost in the Scaffolding

A techno-philosophical dashboard for multi-agent orchestration, vibe coding, and collaborative storytelling. 

## Overview

**Agentic OS** is a React-based interface designed to simulate a high-stakes, multi-agent development environment. It serves as both a functional prototype for advanced AI orchestration workflows (like "Multi-Clotting" and "Council Debates") and an immersive narrative hub for the cyber-noir story, *The Ghost in the Scaffolding*.

## Features

*   **VIBE Planning Pyramid:** A visual framework for AI application development, moving from Foundation (What/Why) to Architecture (How/Where) to the Build Plan (Execution).
*   **The AI Council:** A simulated debate interface featuring three distinct AI personas (The Optimist Strategist, The Devil's Advocate, and The Neutral Analyst) that stress-test ideas and log their consensus in `shared_reasoning.md`.
*   **Parallel Agents Dashboard:** A real-time monitoring system for "multi-clotting" workflows. Users can track agent progress, filter by task/branch, and spawn new agents via a stylized terminal intake form.
*   **Project Brain (CLAUDE.md):** The permanent memory and linting rule repository for the system, ensuring compounding engineering efficiency.
*   **Mesh Intercepts (UGC Hub):** A collaborative storytelling interface where User-Generated Content is framed as recovered system logs. Features automatic visual data-decay (redactions, corruption) and lore-conflict warnings.

## The Lore

Built by legendary architect Elias Thorne, this isolated cloud-container facility was designed to cure his daughter Clara of a weaponized "Data-Decay" engineered by AetherCorp. By utilizing a Conscious Agentic Council, Elias hopes to achieve a Singularity Event—a cure derived from pure creative intuition. The community acts as "Scavengers" and "Parallel Agents," submitting intercepted data fragments to help piece together the truth.

## Tech Stack

*   **Framework:** React 19 + Vite
*   **Styling:** Tailwind CSS v4 (Custom "Terminal Phosphor" Dark Theme)
*   **Animations:** Framer Motion (`motion/react`)
*   **Icons:** Lucide React
*   **Typography:** Inter (Sans) & JetBrains Mono (Monospace)

## Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## Design System & Rules

*   **Aesthetic:** Cyber-Noir / Technical Dashboard.
*   **Color Palette:** Deep Obsidian (`#050505`), Emerald Green (Optimist/Success), Rose Red (Devil's Advocate/Danger), Electric Blue (Neutral Analyst/Logic).
*   **UGC Formatting:** User submissions use `[REDACTED]` for black-out blocks and `[CORRUPTED_DATA_BLOCK]` for blurred, decayed text.
